# Documentation Index

Welcome to the Password Security Tool documentation. Use these pages to learn how to install, run, and extend the project.

## Docs

- [Usage guide](usage.md) - quick CLI commands for password analysis, generation, breach checking, and hash demos.
- [Authentication and RBAC](authentication.md) - registration, login, roles, sessions, OTP flows, and environment configuration.
- [Password analysis engine](password_analysis_engine.md) - strength scoring, entropy calculations, dictionary checks, and sample test cases.

## Project layout

- `src/password_security_tool/` - application source code and Flask web app.
- `tests/` - automated test cases.
- `password_tool.sqlite3` - local SQLite database used for development and demo runs.
- `.gitignore` - ignores generated files and runtime artifacts.
